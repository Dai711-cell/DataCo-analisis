# Reports

Carpeta de informes y figuras generadas.

## Informes principales

- `demand_business_recommendation_report.Rmd`: informe final para empresa sobre demanda diaria.
- `random_forest_arrival_lag_rolling_report.Rmd`: informe empresarial sobre promesas de envio.
- `daily_product_quantity_forecast_report.Rmd`: comparacion tecnica de modelos de demanda.
- `sales_model_comparison_report.Rmd`: modelo inicial de ventas y evidencia de rendimiento.
- `sales_model_lag_rolling_report.Rmd`: prueba de lags/rollings sobre ventas por linea.

## Figuras

Las figuras de `reports/figures/` se mantienen para que los informes Rmd puedan visualizarse al abrirlos localmente o renderizarse a HTML.

Los `.html` renderizados se excluyen del repo porque son pesados y pueden regenerarse desde los `.Rmd`.
